Classpublic class SWFFontDataOutput
InheritanceSWFFontDataOutput Object
Implements IFontDataOutput

Generate a uncompressed SWF (FWS) with one embedded font and it own class Contains the strict minimum of data (no metadata, no MainClass, only FontClass, no background color, stage size [0 x 0], etc.) NOTICE: All tags (except FileAttributes and ShowFrame/End) use the tag length in its extended format NOTICE: Here, DefineFont3 always use flags hasLayout wideoffset and widecode as true NOTICE: Font data could be corrupted, but don't have any security impact since all SWF tags define there length. The only problem could be with Loader.loadBytes() which will never dispatch an Event.COMPLETE event.

fontClassNameproperty
fontClassName:String

Define full qualified class name use to link embedded font to an AS3 class Default auto generated syntax: fstream.fonts.EmbeddedFont$HelveticaNeue85UltraLight_regular where font name is "Helvetica Neue 85 Ultra Light" on font style is "regular" (a flash FontStyle supported) NOTE: The separator of class and its package is "." or "::". Classname or package could contains invalid chars (not supported by ASC - ActionScript Compiler) like symbol, space, null chars, etc. but can produce errors throught getDefinitionByClassName() or applicationDomain.getDefinition() calls.
